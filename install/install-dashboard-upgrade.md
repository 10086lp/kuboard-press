## 执行升级

* 进入 Kuboard 首页，完成登录，进入 Kuboard ***集群概览*** 页
* 点击 ***调整镜像版本*** 按钮
* 勾选 ***监控*** 筛选项，点击刷新
  并修改 Deployment / Kuboard 的镜像版本为您要升级到的目标版本，如下图所示

![image-20190728220831126](./install-dashboard-upgrade.assets/image-20190728220831126.png)

* 点击 ***执行变更***

* 点击 ***应用***

* 点击 ***完成***

  等待容器组调整完毕。

* 刷新浏览器页面
