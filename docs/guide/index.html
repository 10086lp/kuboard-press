# 使用手册

Kuboard 的设计目标

  • 降低 Kubernetes 学习门槛，让初学者先把 Kubernetes 用起来，再逐步理解 Kubernetes 中的各种概念
  • 提高 Kubernetes 运维的便捷性，让资深 Kubernetes 运维人员脱离 kubectl 命令行，直接在 Kuboard 界面中完成日常运维工作
